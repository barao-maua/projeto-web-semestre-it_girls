# 🍎 Doe+ | Plataforma de Doação de Alimentos e Roupas

O **Doe+** é um projeto desenvolvido para a disciplina de [Nome da Disciplina] que visa conectar doadores (pessoas físicas) a ONGs e instituições de caridade de forma ágil e segura. O sistema foca na intermediação de itens essenciais: alimentos (perecíveis e não perecíveis) e vestuário.

---

## 🎯 Objetivo do Projeto
Facilitar o fluxo de doações na região, combatendo o desperdício de alimentos e auxiliando na distribuição de roupas para pessoas em situação de vulnerabilidade, utilizando geolocalização e perfis verificados.
